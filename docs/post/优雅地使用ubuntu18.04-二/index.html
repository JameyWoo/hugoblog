22、添加应用启动图标

安装好pycharm后，没有启动的图标，所以自己设置一个。

首先找到图标文件，链接

然后在/usr/share/applications/ 目录下添加Pycharm.desktop。

修改这个文件。注意，其中的Exec改成执行的命令，Icon改成自己图标的路径。

Icon=/home/jamey/programs/pycharm-2019.1.3/pycharm.ico
Terminal=pycharm
Categories=Pycharm;
